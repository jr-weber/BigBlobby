Detailed Description

Knows how to read program settings from a BlobParamsValidator object and write them to an XML file.

The name of the file to write is obtained from the validator object. The default XML file that is written when the program closes is "data/settings/DefaultBlobParams.xml".

Note that calibration settings and camera-specific settings (such as gain and exposure) are written by CCV to other XML files in the data directory.
