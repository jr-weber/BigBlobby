356 Note on using the Doxygen-generated API from within an SVN repository:

358 If the Doxygen-generated API is added to an SVN repository, the files will be
359 regarded as plain text by default, so in a web browser pointed at a repository
360 file, the html text would be seen, rather than the API in the form of a web
361 page. To make the Doxygen-generated files in the repository show up as a proper
362 API web page, the mime types need to be set when uploaded through SVN.
363 Automatically setting the mime-type can be accomplished by a few minor
364 changes to an SVN config file.
365 
366 If you are using Tortoise SVN on Windows, right-click on your project to get a
367 context menu and then select TortoiseSVN --> Settings. When the Settings dialog
368 box pops open, click on General (should be near the top left of the box), and
369 then click on the Edit button (should be near the bottom right of the box).
370 Clicking on Edit should open up a file named config. You mightm want to make
371 a backup copy of this file before modifying it.
372 
373 If the line "enable-auto-props = yes" is commented out, uncomment it.
374 Then look for the section starting with "[auto-props]". Add the
375 following lines to that section:
376 
377 *.html = svn:mime-type=text/html
378 *.css = svn:mime-type=text/css
379 *.js = svn:mime-type=text/javascript
380 *.txt = svn:mime-type=text/plain;svn:eol-style=native
381 *.png = svn:mime-type=image/png
382 *.jpg = svn:mime-type=image/jpeg
383 *.pdf = svn:mime-type=application/pdf
384 *.jpeg = svn:mime-type=image/jpg
385 *.tiff = svn:mime-type=image/tiff
386 *.tif = svn:mime-type=image/tiff
387 
388 This change will only apply to files as they are commited to the repository,
389 so if you have already committed some Doxygen files, you might need to delete
390 them and then add them back to get the mime type set correctly. If you now
391 point a web browser to the docs/html/index.html file in the SVN repository,
392 you should see the Doxygen Main Page as a web page, rather than the source text.
