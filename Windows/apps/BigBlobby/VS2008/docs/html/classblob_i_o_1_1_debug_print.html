Detailed Description

Utility class for printing debugging messages to standard out.

To provide a convenient way to turn debugging on or off for a class that has an instance of DebugPrint as a data member, either the constructor or the setDebuggingOn() function must be given an argument of true for the print functions to do anything. An argument of false prevents any printing.

This class also has some utility functions that take a primitive type argument (int, float, double, or bool) and return a standard string.
