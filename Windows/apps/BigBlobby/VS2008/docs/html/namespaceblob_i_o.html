Detailed Description

Contains classes for reading/writing program settings as XML files.

The BlobParamsValidator class that is used by the XML reader for data validation is also used as a means of passing that information back to whatever code calls on the read() function of the XML reader class.

The same validator object is also used as a convenient way to pass program settings to the XML file writer class. For reporting errors seen by the reader, writer, and validator, the classes in the blobExceptions namespace are used.

The blobIO namespace also contains a minor utility class called DebugPrint, which is used throughout the program for writing debugging info to standard out.
