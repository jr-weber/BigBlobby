Detailed Description

This utility class knows how to extract info from any subclass of BlobException and format the info for presentation in a modal warning-style QMessageBox.

The message box will display a warning icon next to its message and will have an OK button. The term "modal" as used here means that all user input to the GUI is blocked until the user clicks on the OK button to close the dialog box.
