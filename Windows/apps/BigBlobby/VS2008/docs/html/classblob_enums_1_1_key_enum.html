Detailed Description

Contains an enum for most of the keyboard keys (alphabetic keys and a few special character keys).

This enum was created for passing keyboard event information to the blobCore::BlobCalibration class in a way that would not require that class to know about the Qt GUI. When key press and release events are detected by the blobGUI::CalibrationGLWidget class, the Qt key enum will be converted to this library-independent enum and then passed in to the blobCore::BlobCalibration class, which requires the use of the arrow keys along with several keyboard shortcuts during the touch surface calibration procedure done by CCV 1.3 code.
